Se crear el proyecto "PruebaSD_API" para el Backend.
Se adjunta el script "BBDD" para la creacion de base de datos en SQLServer.
se crea el proyecto "angular-users-app" para la parte del Frontend.


NOTA: ES MUY IMPORATANTE CONTAR CON EL AMBIENTE RESPECTIVO PARA PODER EJECUTAR CADA PROYECTO.
RECUERDE INSTALAR LAS DEPENDENCIAS DE ANGULAR CON EL COMANDO "npm install" PARA SU CORRECTO FUNCIONAMIENTO.
RECUERDE CAMBIAR EL SERVER EN LA API SE ENCUENTRA EN ACTUALMENTE EN "appsettings.json" : 

 "ConnectionStrings": {
     "MiConexion": "Server=FERCHO\\SQLEXPRESS;Database=PruebaSD;Trusted_Connection=True;TrustServerCertificate=True;"
 },
 
 SE DEBE CAMBIAR EL SERVER POR EL CORESPONDIENTE AL EQUIPO.------
